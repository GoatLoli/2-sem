commit - добавление файлов в репозиторий, фиксирование правок
для каждой версии проекта (как бы сканирует и фиксирует текущее
состояние локального репозитория)

push -  отправка изменений в удалённый репозиторий после commit

pull - получение изменений из удалённого репозитория
git pull — связка последующих команд git fetch и git merge
